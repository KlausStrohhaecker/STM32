# STM32 template-project

## How to setup STM32CubeIDE V1.19

### Step #1 . Setup Project

- Check out project

- Start STM32CubeIDE, select new workspace

- File --> Open Projects from File System

- Import Source / Directory : locate project folder and confirm

- Click "Finish"

### Step #2 : Compile the Device Configuration and the Code

- Open the .IOC file (will start the Device Configuration Tool)

- Project --> Generate Code

- Project --> Build All

### Step #3 : Deploy/Debug Code:

- Run --> Run

- Confirm the Launch Configuration setting
